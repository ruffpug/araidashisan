## コマンド

* 依存関係取得
    * `flutter pub get`
* build_runner実行
    * `dart run build_runner build --delete-conflicting-outputs`
* Web向けビルド
    * `flutter build web --web-renderer html`
* テスト実行
    * `flutter test`
* フォーマッタ実行
    * `dart format ./`
* Linter実行
    * `flutter analyze`
